```java
@CrossOrigin
	@PostMapping("/login")
	public boolean callIsLogin(@RequestBody StudentsVO vo, HttpSession httpSession) {
		
		
		boolean isLogin = studentsService.isStudents(vo);
		
		if(isLogin) {
			httpSession.setAttribute("name", "kimminyoung");
		}
		
		return isLogin;
	}
```
로그인 후 callStudentsListByMap 실행 시
