## 역정규화
    역정규화는 일부 쓰기 성능의 손실을 감수하고 데이터를 묶거나 데이터의 복제 사본을 추가함으로써 데이터베이스의 읽기 성능을 개선하려고 시도하는 과정


- 0.데이터설계
- 1.정규화
- 2.역정규화 (선택)
- 3.분산 저장 (선택)
    
    → 데이터가 많을때 A와 B가 데이터를 분산 저장하고, 주기적으로 A와 B를 동기화 해줌(replication)
