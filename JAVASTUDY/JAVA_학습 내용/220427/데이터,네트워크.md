## 데이터를 표현하는 단위
    ***  b(bit) ☞ B(Byte) ☞ KB ☞ MB ☞ GB ☞ TB ☞ PB ☞ EB 

- bit 
```
데이터의 최소 표현 단위
1bit는 '0,1' 2가지만 표현

1bit = 2가지 01
2bit = 4가지 0001
3bit = 8가지 0000 0001
...
nbit = 2^n 
```
```
256bit = 1Byte
1024Byte = 1KB
2^10 = 1KB
2^20 = 1MB
2^30 = 1GB
2^40 = 1TB
2^50 = 1PB
2^60 = 1EB

컴퓨터는 0과1만 인식
```

- Byte
```
8bit == 1Byte == 256(2^8)

영어, 로마자, 숫자 표현하기 위해 Byte가 나옴!
왜 8bit를 8Byte라고 할까?

영어50개 + 숫자10개 + 특수문자66개 + 기타문자(국가코드) = 256
```
```
char x = "a";  //1Byte
영어는 1Byte
한글은 2Byte
```

## 하드디스크 vs 메모리
```
(HDD/SSD)
하드디스크 : 저장 공간(데이터가 영구적으로 남아 있음)
1TB, 100TB

메모리 : 저장 공간(데이터가 날아감)
8GB 16GB 32GB
```
```
하드디스크/메모리 특징
1.데이터가 영구적으로 남아 있냐?
2.저장하는 데이터가 다르다.

메모리 : 길동이는 이클립스와 메모장을 실행했다.
메모리에는 이클립스와 메모장이 저장되어 있다.
```


```
서버는 특정 컴퓨터가 아니라 '누구든지' 서버가 될 수 있다.

고객들이 서비스(DB,web,app)에 접속하려면 서버의 주소를 알아야 함.
주소(IP === 도로명 주소)

일반 컴퓨터는 IP를 1개만 가짐

서비스를 이용하기 위해서 주소와 식별번호를 알아야 한다.
IT에서 주소는 'IP'이고 식별번호는 'PORT(포트번호)'라고 한다.

주소(IP) : 192.168.0.252
식별번호(PORT) : 3306 (MySQL)
=> 192.168.0.252:3306 (DB)
=> 192.168.0.252:8080(Spring)


와이파이(공유기 or 라우터) 
와이파이에서 주는 IP는 내부 아이피(가짜 아이피)라고 한다.
광랜(인터넷 선) -- 라우터 => IP
```
